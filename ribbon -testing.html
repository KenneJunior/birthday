Special Announcement

Diagonal Ribbon Banner

This ribbon is positioned diagonally across the top left corner of the container with 45-degree cuts at both ends, creating a smooth banner effect that touches the inner walls of the container.

The ribbon is created using a rotated rectangle with triangular pseudo-elements to create the diagonal cuts. It remains stable and properly positioned regardless of screen size.

Customize Your Ribbon

Implementation Code

/* HTML Structure */
<div class="container">
    <div class="ribbon-container">
        <div class="ribbon">
            <div class="ribbon-text">Your Text Here</div>
        </div>
    </div>
    <!-- Your content here -->
</div>

/* CSS Styles */
.ribbon-container {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    overflow: hidden;
    pointer-events: none;
    z-index: 2;
}

.ribbon {
    position: absolute;
    top: 30px;
    left: -70px;
    width: calc(100% + 140px);
    height: 50px;
    background: linear-gradient(135deg, #ff6b6b 0%, #ff8e8e 100%);
    transform: rotate(-45deg);
    display: flex;
    align-items: center;
    justify-content: center;
    box-shadow: 0 5px 15px rgba(0, 0, 0, 0.2);
}

.ribbon::before,
.ribbon::after {
    content: '';
    position: absolute;
    top: 0;
    width: 0;
    height: 0;
    border-style: solid;
}

.ribbon::before {
    left: 0;
    border-width: 25px 0 25px 15px;
    border-color: transparent transparent transparent white;
}

.ribbon::after {
    right: 0;
    border-width: 25px 15px 25px 0;
    border-color: transparent white transparent transparent;
}

.ribbon-text {
    color: white;
    font-weight: 700;
    font-size: 1.2rem;
    text-shadow: 1px 1px 3px rgba(0, 0, 0, 0.2);
}
